《万界行者》文字冒险游戏 · 四世界多模式版

原创故事 · 原创角色 · 原创世界观 · 原创游戏机制

MIT 开源许可证 · 允许自由使用、修改、分发，但须保留原始版权声明和来源说明

版本 v1.1.0 · 技术栈：HTML5 + CSS3 + JavaScript（纯前端，无框架依赖）· Emoji 图标（开源）

游戏说明：
四大世界：丧尸围城(12关) / 仙侠大陆(12关) / 星际殖民(13关) / 赛博朋克(14关)，共51关。
关卡模式：战斗 / 解密 / 寻物 / 交流 / 潜行 / 救护 / 生存 / BOSS。
难度系统：首次通关自动普通难度(敌人属性×0.6)，BOSS动态平衡(约10回合击杀)；已通关世界可选普通/困难/噩梦/地狱4个难度，高难度经验加倍。
世界特色装备：每个世界有专属武器、防具、饰品(如青云剑、纳米战甲、量子武士刀等)，战斗中掉落，BOSS必掉2-3件世界装备，确保每世界至少获得一件装备。
掉落系统：补给类道具(回复药)掉落率降低，装备掉落率大幅提升(普通战斗52%、BOSS必掉)；永久属性道具、buff药剂均衡掉落。
天赋系统：8个等级(D/C/B/A/S/SS/SSS/超神)共24个天赋，每世界可抽3次，通关时可保留1个永久天赋，天赋效果可叠加。
装备系统：武器/防具/饰品三大槽位，5种品质(普通/优秀/稀有/史诗/传说)，三大类装备每世界均有特色款。
背包系统：初始20格，可扩展至100格(5级扩容：200/500/1000/2000/5000金币)。支持消耗品、永久属性道具、Buff道具。
属性点系统：每升级获得3点自由属性点，可分配至生命/攻击/防御/速度(每点+3)。
快进模式：开启后剧情跳过打字效果，战斗自动攻击(优先天赋技)。
存档系统：支持多存档 + 自动存档，兼容旧版存档自动补全新字段。
